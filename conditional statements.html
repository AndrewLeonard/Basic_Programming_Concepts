Conditional Statements

These are statements that begin with if and are used to compare variables or request an input from the keyboard or mouse.

There can only be two results for a conditional statement – True or False.

If the result is true then something will happen. If it is false, nothing will happen.

Example:
You are a passenger in a car driving down the road and you come to a roundabout. The driver of the car has forgotten where they are going, so the driver just keeps going round and round the roundabout (an iteration/loop).

Luckily you know where you are going – Armagh.

Each exit from the roundabout has a sign saying where the road leads to.

The first sign says “Newry” (false) so you ignore it.

The second sign says “Portadown” (false) so you ignore it.

The third sign says “Armagh” (true) so you tell the driver to exit the roundabout (the loop) via this road.

Now you continue with your journey (the rest of the program).

 

Basic if Statement


if (conditional statement) {
                Statement block;
}

Code for the above roundabout example

function (roundabout){

var Sign;
var Destination=”Armagh”;

if (Sign==Destination) { // True
            document.write(“This is the correct sign, leave the roundabout at this exit);
 }


}

 

else if statement

Sometimes you want to use a number of if statements after one another to provide outputs, to do this you can use an else if statement.
The else if statement is written like this:

else if

if (conditional statement) {
                Statement block;
}
else if (conditional statement) {
                Statement block;
}

Take the roundabout example above.

Whilst going round and round the roundabout, you looked at each sign and read the destination on it and decided if it matched where you were going.

You could write the code like this:

function (roundabout){

var Sign;
var Destination=”Armagh”;

if (Sign==Destination) { // True
            document.write(“This is the correct sign, leave the roundabout at this exit);
 }
else if (Sign==”Portadown”) { // True
            document.write(“This is not the correct sign, this road leads to Portadown);
 }
else if (Sign==”Newry”) { True
            document.write(“This is not the correct sign, this road leads to Newry);
 }

}

 

else statement

This catches any results that are false.


It is written like this:

if (conditional statement) {
                Statement block;
}
else {
                Statement block;
}

Let’s take the roundabout example above. Instead of an elseif statement for each sign we could just say this not your exit.


function (roundabout){
var Sign;
var Destination=”Armagh”;

if (Sign==Destination) {
            document.write(“This is the correct sign, leave the roundabout at this exit);
 }
else {
            document.write(“This is not the correct sign for your destination, do not  leave the roundabout);

 }

}
