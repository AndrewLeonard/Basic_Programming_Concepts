Variables

These can be assigned a value and used throughout the program.

Their values can be modified as the program progresses or remain static.

 

Types Of Variable

Variables are defined using either text or numbers or a combination of both.

String - Contains text or a combination of text and numbers (alphanumeric).

Integer - A variable assigned a number

Variables can be global or local.

Global variables can be used right across all processes.
Local variables are used within the process they are defined in.

 

In Scratch:

A global variable is one that can be used by all sprites.
A local variable is only used by the sprite it is defined in.

 

Naming Variables

The name of a variable can only contain can contain letters, or a combination of letters and numbers and underscores ( _ ) .

It is important that when you name your variables that you give them names associated with what they are defining. This makes it easier to understand your code.

Example

Temperature of baked beans could be written as:

var a;

but that doesn't really mean a lot.

so we could write it like this:

var TempBakedBeans;

now you now what it means.

 

Naming Convention

There are several conventions, but the two you should be familiar with are these:

1 - First letter of each separate word that make up the variable name are capital letters.
2 - First letter of each separate word is a capital letter with an underscore (_) placed in between each word to join them together.

Example
variable my name could be written as:

var MyName;

or

var My_Name;

 

Using Variables In Code:

When using variables we must define them before they can be used in the program.

Normally you define them at the beginning of the code you are writing.

Example:

var BreadSlices = 2;
var TemperatureOfToast_c = 150;
var QuantityBeans_ml = 80;

Defining A Variable

When defining integer variables, you just write the number after the = sign and then a semicolon (;) after the value.

When defining a string variable, the text must be enclosed with an apostrophe before and after the value and then a semicolon (;) at the end.

Example;

An age can be defined as;

var MyAge = 21;

var MyAge = 'Twenty One';
